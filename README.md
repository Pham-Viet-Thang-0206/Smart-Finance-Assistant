PROJECT - Smart Finance Assistant
Project overview
We will create an app (web app or mobile app) that acts as a smart personal finance assistant. The app helps users track their daily income and expenses conveniently and utilizes the OpenAI API (https://platform.openai.com/docs/introduction) to analyze and optimize personal financial plans.
Features
User authentication: Our app supports OAuth2 API, so users can login using their Google accounts. Users must login to use other app features.
Diverse Input Methods:
Manual Input: Users can manually enter the item name, amount, and category.
Voice Recognition: Use AI to convert voice into expense data (e.g., "Breakfast 30 thousand VND").
QR & Receipt Scanning: Use image recognition technology to extract financial information from receipts or QR codes.
Automated Classification & Analysis (AI-Powered):
Use OpenAI API to automatically categorize expenses into corresponding groups (e.g., Food & Beverage, Transportation, Entertainment).
Compare actual spending with initial financial goals and provide reasonable spending suggestions.
Dashboard:
Shows the activity history, income/expense charts, and financial analysis results of the user.
AI Chatbot : It provides money-saving tips and warnings about overspending based on the user's lifestyle habits.
Requirements
OpenAI API: You will need to obtain an API Key from OpenAI to perform natural language processing, voice recognition, and data analysis tasks.
Front-end: HTML, CSS, JavaScript (React Native), and JSON.
Database:  MySQL
